# UCR-STAT-107-Final-Project
Class: UCR STAT 107

Due: March 19, 2026

Group members: Laura Lu, Iris Shih, Brandon Moore

Project Description: We will analyze the viewer engagement among trending Youtube videos using our dataset, Youtube Videos Statistics 2026 dataset, derived from Kaggle. Our goal is to investigate whether viewer engagement differs across Youtube video categories by calculating engagement based on the likes/dislikes and comments relative to views, visualizing engagement patterns across categories, and fitting a linear regression model to evaluate differences in engagement levels.
